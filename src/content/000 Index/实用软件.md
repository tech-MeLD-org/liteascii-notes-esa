# yt-dlp

GitHub 下载 `yt-dlp_win.zip` 版本

```shell
yt-dlp --version
yt-dlp URL
yt-dlp -f bestvideo+bestaudio URL    #下载最高画质
yt-dlp -x --audio-format mp3 URL	   #只下载音频，需要安装FFmpeg

yt-dlp -F URL	                   #查看所有可下载清晰度
yt-dlp -f ID1+ID2 URL				#下载指定格式

yt-dlp https://youtube.com/playlist?list=xxxx	#下载整个播放列表
```

统一输出为 MP 4

```shell
yt-dlp -f "bv*+ba/b" --merge-output-format webm https://youtube.com/xxxx
```

- `bv*+ba` → 最佳视频 + 最佳音频
- `/b` → 如果没有分离流就下载单文件
- `--merge-output-format mp4` → 合并为 mp4
