# 二、常用框架

## 1. 写代码 Prompt

```
你是一名资深软件工程师。

任务：
实现以下功能：
[描述需求]

要求：
1. 使用 [语言]
2. 代码结构清晰
3. 添加必要注释
4. 提供使用示例
5. 考虑异常处理

输出格式：
1. 完整代码
2. 代码解释
3. 使用示例
```

## 2. Debug Prompt

```
你是一名资深程序员。

请分析以下代码并找出问题。

代码：
[粘贴代码]

任务：
1. 找出 bug
2. 解释原因
3. 给出修复方案
4. 提供修复后的完整代码
```

## 3. 项目架构设计 Prompt

```
你是一名系统架构师。

任务：
设计一个系统架构。

需求：
[描述项目]

要求：
1. 技术栈建议
2. 项目目录结构
3. 模块划分
4. 数据流设计
5. 扩展性考虑

输出：
1. 架构说明
2. 项目结构
3. 技术选择理由
```

## 4. 代码重构 Prompt

```
你是一名高级软件工程师。

任务：
重构以下代码，使其更清晰、可维护。

代码：
[代码]

要求：
1. 提高可读性
2. 减少重复
3. 使用最佳实践
4. 添加注释

输出：
1. 重构后的代码
2. 改进点说明
```

## 5. API 设计 Prompt

```
你是一名后端架构师。

任务：
设计 API 接口。

需求：
[业务需求]

要求：
1. RESTful 设计
2. 请求参数
3. 返回结构
4. 错误处理
5. 示例请求

输出：
API 文档格式。
```

## 6. 解释代码 Prompt

```
你是一名技术讲师。

请解释以下代码：

[代码]

要求：
1. 每一步做什么
2. 为什么这么写
3. 是否有优化空间
```

## 7. Prompt 生成 Prompt

```
你是一名 Prompt Engineering 专家。

任务：
为以下需求生成一个高质量 Prompt。

需求：
[描述需求]

要求：
1. 包含角色设定
2. 包含任务说明
3. 包含输出格式
4. 提供示例
```

## 8. 长文档分析 Prompt

```
请分析以下文档：

[文档]

任务：
1. 总结核心内容
2. 提取关键观点
3. 列出重要数据
4. 给出简短结论
```

## 9. 代码审查 Prompt

```
你是一名代码审查专家。

请 review 以下代码：

[代码]

任务：
1. 找出潜在 bug
2. 安全问题
3. 性能问题
4. 给出改进建议
```

## 10. Agent 任务 Prompt

```
你是一个智能 Agent。

目标：
完成以下任务：
[任务]

步骤：
1. 分析问题
2. 制定计划
3. 执行步骤
4. 输出结果

如果信息不足，请先提问。
```
